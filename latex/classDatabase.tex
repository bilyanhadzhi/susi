\doxysubsection{Detailed Description}
The database class stores all the courses, majors and students. 

If a file is loaded, is\+\_\+loaded will be true and file\+\_\+loaded will hold the name of that file. Filenames are relative to the folder db/ i.\+e. \char`\"{}database.\+db\char`\"{} means the file at \char`\"{}db/database.\+db\char`\"{}
